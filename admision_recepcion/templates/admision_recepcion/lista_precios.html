    Instrucciones - Lista de Precios
    🎯 Objetivo del Módulo

    Gestionar la lista de precios de todos los productos y servicios médicos, con códigos CUPS para RIPS y control de convenios.

    🔍 Funcionalidades de Búsqueda
    • Por nombre: Buscar productos por nombre o código
    • Por convenio: Ver precios específicos según convenio
    • Por categoría: Filtrar por tipo de servicio
    • Por RIPS: Filtrar según tipo de registro RIPS
    • Por precio: Rango mínimo y máximo
    📊 Códigos Importantes

    CUPS: Clasificación Única de Procedimientos en Salud

    RIPS: Registro Individual de Prestación de Servicios

    CUM: Código Único de Medicamentos

    CIE-10: Código de diagnóstico

    IVA: Impuesto sobre las ventas

    Convenios: Precios especiales por empresa

    🏥 Tipos de RIPS
    Consulta Procedimiento Otros Servicios Medicamentos
    💡 Uso del Sistema
    1. Seleccione un convenio para ver precios específicos
    2. Use el filtro de búsqueda para encontrar productos
    3. Los códigos CUPS son fundamentales para generar RIPS
    4. Exporte la lista a Excel para uso externo
    5. Los precios pueden variar según el convenio seleccionado
    Nota importante: Los códigos CUPS son esenciales para la generación automática de RIPS. Los medicamentos requieren código CUM para facturación correcta.
